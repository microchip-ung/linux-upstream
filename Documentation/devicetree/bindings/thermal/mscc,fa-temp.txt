* Microchip Fireant Thermal Sensor

This binding describes thermal sensors that is part of Fireant SoCs.

Required properties:
- compatible : Must be "mscc,fa-temp"
- reg : Address of the temperature sensor registers
- #thermal-sensor-cells : must be 0

Example:

	tmon0: tmon@610508110 {
		compatible = "mscc,fa-temp";
		reg = <6 0x10508110 0xc>;
		#thermal-sensor-cells = <0>;
	};
